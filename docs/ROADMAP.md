# JobTrackr Roadmap

## JT-0001 - Project Bootstrap and Product Foundation

- Establish repository structure.
- Add backend FastAPI foundation.
- Add frontend React foundation.
- Define deterministic job search recommendation contracts.
- Document product scope, architecture, and engineering decisions.

## JT-0002 - Frontend Job Discovery Form and Results UI

- Build skill, role, and location input UI.
- Display generated recommendation results.
- Add external search link cards.

## JT-0003 - Manual Opportunity Saving

- Add manual job opportunity form.
- Add local persistence foundation.
- Track basic application status.

## JT-0004 - Analytics and Reporting Foundation

- Show application status counts.
- Add simple progress reports.
- Prepare export-friendly summaries.
